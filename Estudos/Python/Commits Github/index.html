O que são Commits no GitHub?

Commits são alterações salvas no repositório do GitHub. Cada commit representa um ponto específico na história do projeto, permitindo que os desenvolvedores rastreiem mudanças, revertam para versões anteriores e colaborem de forma eficaz.

Importância dos Commits

  • Rastreamento de mudanças: Permite acompanhar o histórico do projeto.
  • Colaboração: Facilita o trabalho em equipe.
  • Reversão: Possibilita voltar a versões anteriores do código.

Como fazer um Commit

  1. Faça alterações no código.
  2. Use o comando git add . para adicionar as mudanças.
  3. Use o comando git commit -m "Mensagem do commit" para salvar as alterações.
  4. Use o comando git push para enviar as mudanças ao repositório remoto.
